# In2Peta frontend

Next.js 14 (App Router) UI for In2Peta. Design tokens and screen blueprints live in [`../design-system.json`](../design-system.json).

For setup, mock mode, environment variables, and deployment, see the **[root README](../README.md)**.

```powershell
npm install
copy .env.example .env.local
npm run dev
```

Production check: `npm run build`
